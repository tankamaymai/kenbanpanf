# パンフレット画像生成 — 指示書（日本語）

`PAMPHLET-TEXT.md` および `src/*.jsx` で使っている画像スロット向けに、**画像生成 AI**（ChatGPT の画像機能、Imagen など）へ貼り付けられる **日本語プロンプト集**です。

**ツールについて**: 「Imagen 2.0」などモデル名はサービスごとに異なります。利用前に **どの画像モデルか**、**利用規約・医療・広告表現・商用可否**を必ず確認してください。生成結果の品質が安定しない場合は、同じ内容を **英語に意訳**して入力する方法もあります。

**デザインのトーン・色**: `DESIGN.md` 第1〜2節（ティール `#2E8B7A`、清潔感、編集デザイン）に合わせること。

---

## 1. 全カット共通（各プロンプトの末尾に追記推奨）

### 1.1 スタイル固定文（コピペ用）

```
日本の整形外科向け患者説明パンフレット用。清潔で信頼感のある編集デザイン。アクセントは落ち着いたティール・ジェードグリーン（#2E8B7A 付近）のイメージ。白基調、自然光、画面は明るくシャープ。写真指定のカットはリアルな患者教育写真調。図解指定はフラットで分かりやすい医療イラスト調。ホラー要素・過度なグロ・開放創のクローズアップは避ける。患者は特定できないよう顔は映さないかボカす、あるいは顎下で切る。病棟服や無地の私服。画像内に読める文字・ロゴ・病院名・透かしは入れない。キャプション用の文字も画像内に描かない。
```

### 1.2 除外指定（ツールに「ネガティブプロンプト」欄がある場合）

```
文字, ロゴ, 透かし, 過激な出血, 骨の露出, 指が6本, 手指の変形, 解剖の明らかな誤り, 子ども患者, 不快な陰鬱なトーン
```

### 1.3 アスペクト比の目安

| 用途 | 推奨比率 | メモ |
|------|-----------|------|
| 手術ページのワイドヒーロー | **21:9** 〜 **16:9** | 横長 |
| 解剖・説明図 | **4:3** または **1:1** | 線画〜ソフトイラスト |
| 外転枕 OK/NG（1コマ） | **4:5** または **3:4** | 装具全体が収まる |
| 創部・日常シーンのカード | **16:9** | トリミングしやすい |
| 表紙フォト（Cover V2） | **3:4** または **9:16** | 左カラム背景想定 |

**解像度**: 可能なら長辺 **2000px 以上**（印刷・再トリミング用）。Web 掲載前に圧縮。

## 3. ページ別プロンプト

各項目は **①短く**（入力欄が短い場合） **②詳しく**（指示を細かく渡せる場合）。**どちらか一方**をコピーして使う。

---

### 3.1 表紙 Cover V2 背景 `brace-front2.jpg`

**①短く**

```
中年の日本人成人が外転枕（肩外転位保持装具）を装着。正面から、胴〜装具が画面に収まる。背景は院内をぼかし。顔は見せない。患者説明用の落ち着いた写真。
```

**②詳しく**

```
外転枕（大きな腕下枕で腕を体から少し離し、腰・胸部ベルトで固定する装具）を付けた患者の写真。正面、フォトリアル。日本人成人、性別は特定しにくい見せ方。顔は映さないか顎下でカット。白〜ティール系の明るい院内照明。装具全体がはっきりわかる。画像内に文字なし。
```

### 3.2 解剖（P03）`anatomy-front-bone.jpg`

**①短く**

```
肩関節の正面、骨格のみの医療イラスト。肩甲骨と上腕骨が分かる。文字ラベルなし。白背景。パンフレット向けの柔らかい色。
```

**②詳しく**

```
肩の正面から見た骨格の簡潔な解剖図。肩甲骨・鎖骨・上腕骨近位がはっきり。腱板説明用パンフレットのトーン。グレーとティール系の線。画像に語句を書き込まない。印刷向きのクリーンなベクター風。
```

### 3.3 解剖（P03）`anatomy-front-muscle.jpg`（肩甲下筋）

**①短く**

```
肩の正面の医療イラスト。肩甲下筋だけを柔らかいティール色で強調。他の筋は薄く。白背景。文字なし。
```

**②詳しく**

```
①と同系統の正面肩イラスト。肩甲下筋を前面〜肩甲領域で半透明のティール（想像色 #2E8B7A 系）で強調。其余解剖は弱め。ラベル文字は画像内に入れない。
```

### 3.4 解剖（P03）`anatomy-back-muscle.jpg`（棘上・棘下・小円）

```
肩の後ろから見た教育用イラスト。棘上筋・棘下筋・小円筋の3つを、穏やかな色分けで示す。リアルすぎない簡潔さ。臨床パレットにティールのアクセント。タイポグラフィなし。
```

**②詳しく**

```
21対9 の横長。ミニマル侵襲の肩関節鏡シーン。手術チームは無菌衣。創は小さく清潔な印象に。関節鏡・器具は示唆のみで血腥さは出さない。照明は臨床の明るさ。顔は判別不可。「小さな切開・関節鏡」を伝える患者教育にふさわしいトーン。
```

## 4. 英語にした方がよい場合

一部のモデルは **英語プロンプトの方が破綻が少ない**ことがあります。各節の **②詳しく** を機械翻訳して試すか、要点だけ英語のキーワード列（`shoulder abduction pillow`, `patient education`, `no text`, `teal accent` など）を追加してください。

## 5. 権利・安全・運用上の注意

- 実在の機器名・ロゴ・術式映像に **酷似**させすぎない。汎用の見た目にとどめる。
- **実臨床の写真**は、同意取得済みの院内向け撮影素材を最優先する運用が無難。生成画像は図解・イメージ補助に使う形がおすすめ。
- 印刷・配布前に **院内科・広報の承認** を得ること。
