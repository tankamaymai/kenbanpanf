# Design System: 腱板断裂 術後パンフレット（新小文字病院 整形外科）

**プロジェクト ID:** なし（ローカル `pamphlet.html` / React コンポーネント構成）

**ソース・オブ・トゥルース:** 本書は `pamphlet.html` が読み込む `assets/css/colors_and_type.css` と `pamphlet.css` のトークン・コンポーネント定義をセマンティックに要約したもの。新規ページ・コンポーネント・印刷調整を行う前に必ず併読すること。

---

## 1. Visual Theme & Atmosphere

**病院資料としての清潔感と安心感**を軸に、情報は高密度でも視線誘導が明確な**編集デザイン（エディトリアル）**に寄せたパンフレット。スクリーン上はグレー背景に白い **A4 カード（紙面）**が縦に並ぶ構成で、**「印刷物をプレビューしている」**質感。

- **ブランドアクセント**は `pamphlet.css` で **医療系ティール／ジェイド**（`#2E8B7A` 系）に統一。`colors_and_type.css` のデフォルト青（K ロゴ由来 `#1F6FD0`）は **本パンフレット上では上書き**される点に注意。
- **影**は控えめ。ページカードはごく薄い浮き上がり（`0 6px 28px rgba(15,23,42,0.08)`）。病棟生活ページなど一部は **アイスブルーのグラデーション背景**（`ward-page`）で場面転換。
- **トーン**: 落ち着き・信頼・読みやすさ優先。装飾は **番号帯・上縁アクセント・丸バッジ・ソフトカード罫・コラム区切り**で機能的に使う（左の太縦ラインは使用しない）。

## 2. Color Palette & Roles

### ブランド（ティール — `pamphlet.css` の `--brand-*` が実効）

| 名称（意味） | Hex | 役割 |
|-------------|-----|------|
| **ジェイド・プライマリ** | `#2E8B7A` | 章番号帯、コールアウト上縁グラデ、丸アイコン、プライマリボタン、成功／情報の主色 |
| **プライマリ・ホバー** | `#246B5E` | インタラクション時の濃化 |
| **プライマリ・プレス** | `#1B5049` | 押下時のさらに濃い状態 |
| **プライマリ・ソフト** | `#EAF4F2` | 見出し背景、コールアウト背景、写真プレースホルダ下地、解剖図ラッパー |
| **プライマリ・エッジ** | `#A8D5CE` | 枠線・フッター区切り・控えめな区切り |

※ `--success` / `--info` 系もプライマリティールに揃え、**OK・推奨・注意喚起（情報）**の肯定側に使う。

### 中立・テキスト（`colors_and_type.css` のニュートラル階層）

- **見出し・強調本文**: `--neutral-900`（`#0F172A`）に近いダークスレート
- **本文**: `--fg-1`（`#1F2937` 相当の primary text）
- **補助**: `--fg-2`, `--fg-3`（グレー段階）
- **区画背景**: `--neutral-50`〜`100`（KV リスト、sleep テキストボックスなど）

### セマンティック（状態）

| 名称 | Hex | 役割 |
|------|-----|------|
| **警告アンバー（ルール・汎用）** | `#C27A1A` 前後（`--warning` / コールアウト warn） | 注意・要確認 |
| **ストップレッド** | `#C4392D`（`--danger`） | NG・禁止マーク、危険コールアウト |
| **ソフトレッド背景** | `#FBE8E6` | 禁止系コールアウト床 |
| **注意イエロー系** | 例: `#FFF8DC` 背景 + `#E8C84A` 枠、`#7A5F00` 字（`dress-point` 等） | 強い注意バナー、着脱コラムのポイント |
| **再断裂ページ（P4）** | ページ全体クラス `.p4-page`：纸张グラデ／エディトリアル2カラム。警告は **落ち着いたトーン** のカード（`.p4-key-message`）に統一。旧 **原色赤ボックス（`p4-alert-box`）は廃止** |
| **外転枕 NG コーラル** | `#E05A6A` | `okng-col.ng` 等の代替 NG アクセント |

### 画面スケルトン

- **アプリ背景（画面モード）**: `#E5E7EB`（`html, body`）
- **紙面（ページ）**: `#fff`、印刷時は白基調で `@page A4`

## 3. Typography Rules

### フォント方針

- **日本語本体**: **UD デジタル教科書体**（TypeSquare、契約 ID は `colors_and_type.css` 記載）。オフライン・未設定時は **BIZ UDPGothic** がフォールバック。
- **欧文・数字・アクロニム**: **Inter**（`--font-sans-latin`）。章キッカー、ページ番号、ROM 等のラベルに使用。
- **明朝**: `BIZ UDPMincho`（`--font-serif-ja`）— 必要箇所のみ。

### 日本人向け組版

- `line-break: strict`、`word-break: normal` / 見出しは `auto-phrase`、`overflow-wrap: anywhere`
- 本文 **letter-spacing は約 0.01em** — 詰めすぎない
- **行間**: パンフレット実効 `--body-lh: 1.8`、`--body-size: 17px`（高齢患者可読性優先）

### 階層（代表的サイズ — コンポーネントにより変動）

- **表紙タイトル**: 38〜46px 相当、太め（700〜900）、やや字詰め〜均一トラッキング
- **ページ見出し（`.page-head .title`）**: 22px, weight 800
- **英字キッカー**: 10〜12px, `uppercase`, `letter-spacing: 0.2em` 前後
- **サブ見出し（`.subhead`）**: 17px, ソフトティール〜白のグラデ背景 + 淡い枠・角丸（左縦バーは使わない）
- **リード**: 本文 +0.5px 程度
- **キャプション・フッター**: 11〜13px, `--fg-3` 寄り

## 4. Component Stylings

### 紙面（`.page`）

- **スクリーン**: 幅 **794px**、高さ **1123px**（**`min-height` / `max-height` も 1123px** で A4 の縦を固定、`overflow: hidden` で1枚に収める）。内側余白 **50px 58px 62px**、角はシャープ、**ごく薄いドロップシャドウ**
- **本文エリア（`.page-content`）**: 基本 **`overflow-y: auto`**（はみ出し時のみ**紙の内側**で縦スクロール）。**印刷**（`@media print`）と **`pamphlet-print.html`** では **`overflow: hidden`**。子の段間 `gap` は **10px** 前後。
- **印刷**: `210mm × 297mm`、余白は mm 指定に再マップ、`box-shadow` なし、`page-break-after: always`（最終ページ以外）

### ページヘッダー（`.page-head`）

- **左**: 大号数・白字 on **プライマリ塗り**、角は左のみ 6px
- **右**: **ソフトティール**背景にタイトル。**1.5px エッジ色**の枠で帯状

### フッター（`.page-foot`）

- 下部固定、**エッジ色の上罫線**、中央寄せ風ページ番号（欧文太字 + 左右インセット細線）

### トップバー（画面のみ）

- **スティッキー**、半透明白 + **ブラー**、下辺ヘアライン。ロゴ + タイトル + **ピル型**アウトライン／プライマリボタン

### コールアウト（`.callout`）

- **上縁 3px のトーン別グラデ** + 淡い全体枠・角丸 + タイトル先頭ドット（ソフトな輪郭）。**default / ok / warn / no / info** で背景・上縁・枠・タイトル色が変化。`.support-callout` は従来どおりエディトリアル枠のみ（上縁バーなし）。バッジ「i」は非表示設計。

### OK / NG（`.okng` / `.okng-card` ほか）

- **2 カラムグリッド**、各カードは **ラウンド（`--radius-md`）**、ヘッダーバッジに **丸アイコン**。OK はティール系、NG はレッド系。**`tweak-okng-bordered` / `tweak-okng-stamp`** で別バリエーション。

### ルールリスト（`.rule-list`）

- **丸マーク**（28px）に ◎○△✗ 的な意味。**下線は破線**（`border-dashed`）

### タイムライン・チェックリスト・KV・エクササイズカード

- タイムライン: **縦ラインはエッジ色**、ドットはプライマリ + 白リング
- チェックリスト: **ティール枠の角丸チェック** + ✔
- `.ex-card`: 左 **固定幅フォト列**（プレースホルダ + QR 装飾）、右に本文
- `.kv`: 薄グレー背景 **10px ラウンド**、dt/dd グリッド。**手術ページ（`.p5-kv`）**はティール系グラデ枠で上書き

### 表紙バリエーション（`.cover-v1` / `v2` / `v3`）

- **v1 クラシック**: ロゴ列 + 太いアイライン + 大タイトル
- **v2 フォト主役**: 上段ヒーロー（暗幕 + 青系グラデーションオーバーレイ）、下段タイポ
- **v3 エディトリアル**: 左 **ソフトティール wide カラム** + 特大欧文数字、右に TOC・タイトル

### 特例ページ（CSS 接頭辞）

- **`p4-*` / `.p4-page`**: 再断裂章 — エディトリアル2カラム、手術写真枠、リスク3列カード（図版スペース）等。旧デザインの黄バナー・原色赤帯は廃止。
- **`p5-*` / `.p5-page`**: 手術章 — リードの次に **大きめの手術中ヒーロー**（`.p5-intraop-*`、`aspect-ratio` や `min-height` で従来より高く、下に注釈）。その下は **`.p5-lower-stack`** で縦並び：**要点**（`.subhead` + `.p5-kv`）と **コールアウト**を先に置き、**一番下に創部の `PhotoCard`（ワイド 16:9 想定）**で締める。創部カードは **上縁ティールのアクセント**＋エディトリアル枠。`.p5-kv` はティール系グラデ枠。紙面は P4 と同系のごく淡いグラデ背景。
- **`pX-*` / `okng-layout`**: 外転枕 — 太い OK/NG ヘッダ、3 枚フォトストリップ
- **`brace-wear-rules`**: 装着中ルール — **1 枚の角丸カード**内に「守ること／やってはいけないこと」を **中央の細いティール区切り**で分割。床はソフトティールのグラデ、行は破線区切り。OK アイコンはティール系、NG はコーラル／ソフトレッド系（`.brace-wear-rules-icon--ok` / `--ng`）
- **`ward-*`**: 病棟生活 — アイスブルー页面、ステップ列、動画ボックス + QR
- **`photo-hero` / `photo-steps` / `photo-duo` / `photo-layout`**: 写真主役レイアウト（16:9 ヒーロー、4 列ステップ番号バッジ等）

### 開発用（本番印刷では非表示）

- **`.tweaks-panel` / `.tweaks-toggle`**: 固定配置のバリエーション切替。`@media print` で消去。

## 5. Layout Principles

- **1 画面＝1 A4** のメンタルモデル。要素は **最大幅 ~38em** の読みやすい行長を意識（`.lead`, `.body p`）
- **段組**: `.grid-2` / `.grid-3` / `.grid-a45`、OK/NG・QA は **2 カラム**（狭幅では 1 列に落ちるものあり）
- **余白**: セクション間は **16〜22px** 前後が多い。見出し直後は詰め、**絶対配置フッター**と本文が干渉しないよう `.page` 内パディングで確保
- **印刷**: `break-inside: avoid` をコールアウト、カード、タイムラインステップ等に付与。見出し直後の孤立防止

## 7. AI / デザイン作業向けメモ

- 新しい見出し・注記・カードを追加するときは **既存クラス（`.subhead`, `.callout`, `.page-head` 等）を優先**し、色は **CSS 変数**で足す。
- **ブランド色を青に戻さない**（本パンフレットはティール上書きが正）。
- Stitch 等へプロンプトする場合も、**「ジェイド系プライマリ #2E8B7A + ソフトミント背景 #EAF4F2 + 白紙面 + A4 編集トーン」**と **hex** をセットで渡すとブレにくい。
